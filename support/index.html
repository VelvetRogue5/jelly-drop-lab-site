What To Include

Please include your app version, device model, iOS version, level number, and a short description of the issue. Screenshots or screen recordings are helpful when a visual effect, level state, or crash is involved.

Common Questions

If a level seems blocked, try undoing the last move, restarting the level, or using an available helper such as an extra tube. If the issue repeats on the same level, report the level number so we can review the puzzle setup.

If sound is missing, check the in-game sound toggle, the device mute switch, and the device volume.
